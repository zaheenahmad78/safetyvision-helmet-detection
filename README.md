# 🪖 SafetyVision AI - Construction Site Safety Monitor

Real-time helmet detection system for construction sites using YOLOv8 and Streamlit.

## 🚀 Live Demo
Screenshot 2026-03-31 210414.png
**Try it now:** [SafetyVision AI Live App](https://safetyvision-helmet-detection-snexrn4ojvvymvx9yjin2e.streamlit.app/)

## 📋 Features

- Upload construction site images
- AI detects helmets and workers in real-time
- Shows helmet count and person count
- Highlights workers without helmets
- Side-by-side original vs detection view

## 🛠 Tech Stack

- **Python 3.11**
- **YOLOv8** (Ultralytics) - Object Detection
- **Streamlit** - Web Interface
- **OpenCV** - Image Processing
- **Streamlit Cloud** - Deployment
